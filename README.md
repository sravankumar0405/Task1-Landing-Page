# 👟 Walkwave - Future of the Sports

Walkwave is a modern, high-performance, and fully responsive e-commerce landing page designed for a next-generation athletic footwear brand. Built using modern semantic HTML5 structures, customized external CSS configurations, and the latest dynamic **Tailwind CSS v4** engine.

---

## 🚀 Features

* **Modern Hero Section:** Features an aggressive, bold typography layout matching premium athletic brands, complemented by high-impact CTA buttons and smooth interactive image transitions.
* **Fully Responsive Architecture:** Uses a mobile-first design strategy implementing Tailwind breakpoints (`sm:`, `md:`, `lg:`) to provide an optimized experience across mobile devices, tablets, and desktop displays.
* **Interactive Mobile Navigation:** A lightweight, pure JavaScript-powered hamburger drawer overlay tailored specifically for mobile viewports.
* **Tailwind CSS v4 Integration:** Leverages the updated play engine configuration directly inside the document head for immediate dynamic styling.
* **Custom Typography Layering:** Seamless implementation of external Google Fonts (`Oswald` and `Figtree`) to distinctly separate high-profile athletic display headers from highly readable UI prose.

---

## 🛠️ Tech Stack & Architecture

* **HTML5:** Structured using clean, modern semantic conventions.
* **Tailwind CSS v4:** Leverages utility-first styling patterns, background color linear interpolations, dynamic aspect ratio controls, and scale animations, and used Prebuilt.UI for adding Tailwind CSS components some sections.
* **Custom CSS:** Handles refined component adjustments, layout overrides, and external font configurations (`style.css`).
* **JavaScript (ES6):** Used to toggle DOM element state classes to execute smooth responsive menu layouts.
* **AI Tools:** Used AI tools like Google Gemini, and ChatGpt for adjusting some responsive issues for this project.
